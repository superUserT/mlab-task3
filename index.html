üì° API Endpoints

Movies Endpoints

GET /movies

Returns all movies in the library.

Example:

curl http://localhost:3000/movies

POST /movies/:title

Adds a new movie or updates an existing one.

Example:

curl -X POST http://localhost:3000/movies \
  -H "Content-Type: application/json" \
  -d '{"title": "New Movie"}'

Series Endpoints

GET /series

Returns all series in the library.

Example:

curl http://localhost:3000/series

Songs Endpoints

POST /songs

Adds a new song (requires name, artist, and year).

Example:

curl -X POST http://localhost:3000/songs \
  -H "Content-Type: application/json" \
  -d '{"name": "New Song", "artist": "Artist Name", "year": 2023}'

üîí Notes

  • The API uses file-based storage (JSON files in the json_files directory).
  • All endpoints support standard CRUD operations (GET, POST, PUT, DELETE).
  • Error handling is implemented for common scenarios.
